# TEMPLATE_RUN

Copy this folder to `governance/evaluation-inputs/<run-id>/` when starting a new evaluation cycle.

## Minimum expected outputs
- `01_manifests/` : run + dataset manifests
- `02_runs/baseline/` and `02_runs/candidate/` : metrics + latency/cost + compliance
- `03_governance/comparison/` : normalized deltas + recommendation input
- `03_governance/gate/` : final gate decision and risk notes
